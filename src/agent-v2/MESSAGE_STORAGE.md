## 三、对比总结

| 层级 | 当前实现 | OpenClaw | 说明 |
|------|----------|----------|------|
| **内存层** | MessageList | Agent 内存 | 类似，实时读写 |
| **缓冲层** | MessageBuffer | 无 | 当前独有，批量优化 |
| **会话层** | SessionManager | Session Transcripts | 类似 |
| **持久层** | FileStorage (JSON) | FileStorage (Markdown) | OpenClaw 更适合阅读 |
| **索引层** | 无 | SQLite FTS | OpenClaw 支持搜索 |
| **Memory** | 无 | Daily + Long-term | OpenClaw 更完善 |

## 四、消息状态生命周期

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                        消息生命周期                                             │
├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                 │
│   创建阶段                                                                      │
│   ─────────                                                                    │
│                                                                                 │
│   1. addAssistantMessage({ content: '' })                                     │
│      ┌─────────────────────────────────────────────────────────────────────┐    │
│      │ 内存: MessageList._messages.push(message)                          │    │
│      │ 状态: 'streaming'                                                  │    │
│      └─────────────────────────────────────────────────────────────────────┘    │
│      │                                                                        │
│      │ messageBuffer.add(message)                                           │
│      ▼                                                                        │
│   ┌─────────────────────────────────────────────────────────────────────┐      │
│   │ 持久化: FileStorage.saveMessage()                                    │      │
│   │ 状态: 'created'                                                      │      │
│   └─────────────────────────────────────────────────────────────────────┘      │
│                                                                                 │
│   流式更新阶段                                                                  │
│   ─────────────                                                              │
│                                                                                 │
│   2. for chunk of response:                                                  │
│      ┌─────────────────────────────────────────────────────────────────────┐    │
│      │ 内存: messageList.updateMessageById({ content: text })              │    │
│      │ 增量: content += chunk.content                                     │    │
│      └─────────────────────────────────────────────────────────────────────┘    │
│      │                                                                        │
│      │ messageBuffer.update(messageId, { content })                         │
│      ▼                                                                        │
│   ┌─────────────────────────────────────────────────────────────────────┐      │
│   │ 持久化: FileStorage.updateMessage()                                 │      │
│   │ 状态: 'streaming'                                                  │      │
│   └─────────────────────────────────────────────────────────────────────┘      │
│                                                                                 │
│   完成阶段                                                                      │
│   ─────────                                                                    │
│                                                                                 │
│   3. finish_reason === 'stop':                                               │
│      ┌─────────────────────────────────────────────────────────────────────┐    │
│      │ 内存: updateMessageById({ finish_reason, usage })                   │    │
│      └─────────────────────────────────────────────────────────────────────┘    │
│      │                                                                        │
│      │ messageBuffer.update(messageId, { finish_reason, usage })            │
│      ▼                                                                        │
│   ┌─────────────────────────────────────────────────────────────────────┐      │
│   │ 持久化: FileStorage.updateMessage()                                 │      │
│   │ 状态: 'completed'                                                   │      │
│   └─────────────────────────────────────────────────────────────────────┘      │
│                                                                                 │
│   工具调用阶段 (如有)                                                        │
│   ───────────────────                                                       │
│                                                                                 │
│   4. processToolCalls():                                                     │
│      ┌─────────────────────────────────────────────────────────────────────┐    │
│      │ 内存: addToolResult()                                             │    │
│      └─────────────────────────────────────────────────────────────────────┘    │
│      │                                                                        │
│      │ messageBuffer.add(toolResult)                                       │
│      ▼                                                                        │
│   ┌─────────────────────────────────────────────────────────────────────┐      │
│   │ 持久化: FileStorage.saveMessage()                                    │      │
│   └─────────────────────────────────────────────────────────────────────┘      │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘
```

## 五、数据一致性保证

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                        数据一致性保证                                          │
├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                 │
│  1. 内存一致性                                                                │
│     ┌─────────────────────────────────────────────────────────────────────┐    │
│     │ MessageList 使用互斥锁 (Promise 队列)                              │    │
│     │ - 确保同时只有一个操作修改消息列表                                 │    │
│     │ - addAsync() / updateMessageByIdAsync()                           │    │
│     └─────────────────────────────────────────────────────────────────────┘    │
│                                                                                 │
│  2. 持久化一致性                                                              │
│     ┌─────────────────────────────────────────────────────────────────────┐    写入顺序保证 │
│     │                                                        │    │
│     │ - 先写内存 → 再写缓冲区 → 最后持久化                               │    │
│     │ - 内存始终是最新的                                                 │    │
│     │                                                                   │    │
│     │ 崩溃恢复                                                            │    │
│     │ - Step 完成时保存快照                                              │    │
│     │ - Agent 停止时强制刷新                                            │    │
│     │ - 重启后可从快照恢复                                               │    │
│     └─────────────────────────────────────────────────────────────────────┘    │
│                                                                                 │
│  3. 消息不丢失                                                                │
│     ┌─────────────────────────────────────────────────────────────────────┐    │
│     │ 三层保护                                                            │    │
│     │ - 保护1: 先创建空消息，确保 ID 存在                              │    │
│     │ - 保护2: 每个 chunk 都实时更新                                     │    │
│     │ - 保护3: 多个持久化时机                                           │    │
│     │   * chunk 到达: 缓冲区                                            │    │
│     │   * step 完成: 快照                                               │    │
│     │   * agent 停止: 最终保存                                          │    │
│     └─────────────────────────────────────────────────────────────────────┘    │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘
```
