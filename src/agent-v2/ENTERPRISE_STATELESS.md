## 二、无状态 Agent 核心设计

```
┌─────────────────────────────────────────────────────────────────────────────────────────────┐
│                                无状态 Agent 核心                                              │
├─────────────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                              │
│  ┌───────────────────────────────────────────────────────────────────────────────────────┐   │
│  │                              输入 (Input)                                             │   │
│  │                                                                                       │   │
│  │   interface AgentInput {                                                             │   │
│  │       conversationId: string;           // 对话链 ID                                  │   │
│  │       messages: Message[];              // Context 消息 (外部传入)                   │   │
│  │       systemPrompt?: string;           // 系统提示                                   │   │
│  │       tools?: Tool[];                   // 可用工具                                   │   │
│  │       config?: LLMConfig;               // LLM 配置                                   │   │
│  │       maxSteps?: number;                // 最大步数                                   │   │
│  │       metadata?: Record<string, any>;  // 元数据                                     │   │
│  │   }                                                                                   │   │
│  │                                                                                       │   │
│  └───────────────────────────────────────────────────────────────────────────────────────┘   │
│                                                    │                                        │
│                                                    ▼                                        │
│  ┌───────────────────────────────────────────────────────────────────────────────────────┐   │
│  │                              处理流程                                                 │   │
│  │                                                                                       │   │
│  │   for (step = 0; step < maxSteps; step++) {                                        │   │
│  │                                                                                       │   │
│  │       // 1. 构造 LLM 请求                                                            │   │
│  │       const request = buildLLMRequest(input.messages, input.config);                │   │
│  │                                                                                       │   │
│  │       // 2. 调用 LLM                                                                │   │
│  │       const response = await llm.generate(request);                                 │   │
│  │                                                                                       │   │
│  │       // 3. 添加助手消息                                                             │   │
│  │       input.messages.push(response.message);                                       │   │
│  │                                                                                       │   │
│  │       // 4. 处理工具调用                                                             │   │
│  │       if (response.toolCalls.length > 0) {                                        │   │
│  │           for (const call of response.toolCalls) {                                 │   │
│  │               const result = await executeTool(call);                              │   │
│  │               input.messages.push(result);                                         │   │
│  │           }                                                                         │   │
│  │           continue;  // 继续下一轮                                                   │   │
│  │       }                                                                               │   │
│  │                                                                                       │   │
│  │       // 5. 无工具调用，返回结果                                                     │   │
│  │       break;                                                                          │   │
│  │   }                                                                                   │   │
│  │                                                                                       │   │
│  └───────────────────────────────────────────────────────────────────────────────────────┘   │
│                                                    │                                        │
│                                                    ▼                                        │
│  ┌───────────────────────────────────────────────────────────────────────────────────────┐   │
│  │                              输出 (Output)                                            │   │
│  │                                                                                       │   │
│  │   interface AgentOutput {                                                            │   │
│  │       messages: Message[];              // 包含新增消息                             │   │
│  │       finishReason: 'stop' | 'max_steps' | 'error';                                │   │
│  │       usage?: Usage;                     // Token 使用量                            │   │
│  │       steps: number;                     // 执行的步数                              │   │
│  │   }                                                                                   │   │
│  │                                                                                       │   │
│  └───────────────────────────────────────────────────────────────────────────────────────┘   │
│                                                                                              │
│  ┌───────────────────────────────────────────────────────────────────────────────────────┐   │
│  │                              关键特性                                                  │   │
│  │                                                                                       │   │
│  │   ✅ 无状态: 不存储任何会话信息                                                       │   │
│  │   ✅ 幂等性: 相同输入产生相同输出                                                     │   │
│  │   ✅ 可重入: 支持断点续传 (通过 messages 参数)                                        │   │
│  │   ✅ 可测试: 只需 mock LLM 和 Tools                                                   │   │
│  │                                                                                       │   │
│  └───────────────────────────────────────────────────────────────────────────────────────┘   │
│                                                                                              │
└─────────────────────────────────────────────────────────────────────────────────────────────┘
```

│  │   └───────────────────────────────────────────────────────────────────────────────┘ │   │
