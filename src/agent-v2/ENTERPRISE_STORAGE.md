## 七、技术选型总结

| 层级 | 组件 | 理由 |
|------|------|------|
| **实时缓存** | Redis Cluster | 高性能、持久化、集群 |
| **消息队列** | Kafka | 高吞吐、持久化、顺序性 |
| **OLAP** | ClickHouse | 列式存储、聚合查询快 |
| **对象存储** | MinIO/S3 | 成本低、容量无限 |
| **向量存储** | Milvus | 成熟、支持海量向量 |
| **全文搜索** | Elasticsearch | 功能强大、生态完善 |
| **元数据** | PostgreSQL | 事务支持、复杂查询 |
| **向量检索** | pgvector | 简单场景替代 Milvus |
